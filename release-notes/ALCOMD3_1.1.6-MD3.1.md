# ALCOMD3 v1.1.6-MD3.1

## English

ALCOMD3 is a downstream fork of upstream ALCOM. It keeps the original VPM repository, package management, project management, and template features, with the following differences:

### Differences from upstream ALCOM

- Uses a Material Design 3 style interface.
- Adds Material Theme settings, including custom app theme colors.
- Renames the visible app name, window title, installer, shortcuts, and icon branding to ALCOMD3.
- Adds VPM repository reordering to adjust repository priority.
- Adds the ability to view the package list contained in each repository.
- The Windows installer is named with the ALCOMD3 brand and version number, for example:

```text
alcomd3-1.1.6-MD3.1-setup.exe
```

### Installation Notes

The Windows version replaces the original ALCOM installation instead of installing side by side.  
The main executable remains `ALCOM.exe` for compatibility with the existing install path and protocol associations.

## 日本語

ALCOMD3 は、上流版 ALCOM をベースにした下流フォークです。元の VPM リポジトリ、パッケージ管理、プロジェクト管理、テンプレート機能はそのまま維持し、主に以下の点が異なります。

### 上流版 ALCOM との違い

- Material Design 3 風のインターフェースを採用しました。
- Material Theme 設定を追加し、アプリのテーマカラーをカスタマイズできるようにしました。
- 表示上のアプリ名、ウィンドウタイトル、インストーラー、ショートカット、アイコンのブランド表記を ALCOMD3 に変更しました。
- VPM リポジトリの並び替え機能を追加し、リポジトリの優先順位を調整できるようにしました。
- 各リポジトリに含まれるパッケージ一覧を確認できる機能を追加しました。
- Windows インストーラーは、ALCOMD3 の名前とバージョン番号を含む形式になりました。例：

```text
alcomd3-1.1.6-MD3.1-setup.exe
```

### インストールについて

Windows 版は、元の ALCOM と並行してインストールされるのではなく、既存の ALCOM インストールを置き換えます。  
既存のインストール先やプロトコル関連付けとの互換性のため、メイン実行ファイル名は `ALCOM.exe` のままです。

## 中文

ALCOMD3 是基于上游 ALCOM 的下游分支，保留原有的 VPM 仓库、包管理、项目管理和模板功能，主要区别如下：

### 与上游 ALCOM 的区别

- 采用 Material Design 3 风格界面。
- 新增 Material Theme 设置，可自定义应用主题色。
- 应用名称、窗口标题、安装器、快捷方式和图标已改为 ALCOMD3。
- 新增 VPM 仓库排序功能，方便调整仓库优先级。
- 新增查看仓库所包含包列表的功能，便于确认某个仓库提供了哪些包。
- Windows 安装包改为带版本号的 ALCOMD3 命名，例如：

```text
alcomd3-1.1.6-MD3.1-setup.exe
```

### 安装说明

Windows 版本会覆盖原 ALCOM 安装项，而不是并排安装。  
主程序文件名仍保留为 `ALCOM.exe`，用于兼容原有安装路径和协议关联。
